Showcase IDX is a map-centric real-estate property search. This plugin allows Showcase to be embedded into Wordpress. 

== Description ==

Showcase IDX for WordPress is a state-of-the-art, map-centric real estate search platform. You must first
be a member of an MLS and must have access to IDX data in order to use our service. Additionally, a subscription to Showcase IDX is required. Please visit [showcaseidx.com](http://showcaseidx.com) to setup your account. Once you have an API key, simply enter it into the top-level "Showcase IDX" admin menu.


== Installation ==

Once you have an account with Showcase IDX, activate the Showcase IDX plugin and enter the API key.

= WordPress.org Installation =
* Install and activate the "Showcase IDX" plugin from the admin panel.
* Enter API Key in the top-level "Showcase IDX" menu.

= Manual Installation =
* Place the "showcaseidx" folder into the '/wp-content/plugins/' directory.
* Activate "Showcase IDX" plugin.
* Enter API Key in the top-level "Showcase IDX" menu.

== Frequently Asked Questions ==
You will need to be a member of an MLS board that provides IDX data, and have access to that data.
